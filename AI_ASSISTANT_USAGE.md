The project was built with assistance from Anthropic's Claude throughout the development process. 

Initially, I created a rough draft document (included in the GitHub repository) outlining the problem description, desired features of the system, required components for the system to work, key implementation considerations, and potential future improvements. 

After completing this document, I provided the information to the AI, and requested different approaches for tackling each component, along with their respective advantages and disadvantages. Based on this analysis, I identified the most suitable approach for my assignment. 

I then asked the AI to assist me with the code generation of the system as well as to guide me with the deployment process both locally and in the cloud environment. Throughout the entire process, I also used the AI to assist me with debugging code and resolving errors that I was struggling to fix on my own. 

Finally, I asked the AI to generate example requests, visual figures, and tree-like structures/diagrams for my README file to increase its overall clarity and ease of understanding as well as to help me explain, refine, and clarify sections of the project documentation when necessary.
